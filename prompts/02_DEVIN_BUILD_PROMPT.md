# Prompt 02 — Devin Build Prompt

Act as a senior frontend engineer.

## Goal
Implement a fast, polished multi-page website for Jimsolutions Technology that can be deployed to Vercel.

## Constraints
- No Bootstrap
- No jQuery
- No over-animated UI
- No fake certifications or fake awards
- Sample testimonials and sample project descriptions may be used as draft content only

## Deliverables
- Complete page set
- Shared layout
- Reusable sections
- Responsive navigation
- WhatsApp CTA system
- SEO-ready metadata
- Schema markup
- Polished footer

## Quality bar
The result should feel like a real professional small-business website, not a demo.
