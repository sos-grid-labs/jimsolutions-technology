# Prompt 01 — Cursor Build Prompt

Build the Jimsolutions Technology website using the docs in this folder.

## Instructions
- Use Next.js 15 App Router with TypeScript
- Use Tailwind CSS v4
- Use Font Awesome for all icons
- Use Google Fonts for typography
- Use Framer Motion only for subtle section reveals and hover states
- Use images from `public/img`
- Create pages: Home, About, Services, Projects, Gallery, FAQ, Contact
- Make WhatsApp the primary CTA
- Add a floating WhatsApp button globally
- Add click-to-call buttons where appropriate
- Add SEO metadata and schema
- Keep all code readable and modular

## Output
Create the project structure, components, and pages needed for a polished launch-ready website.
