# AGENTS.md

This project is a professional marketing website for Jimsolutions Technology.

## Goal
Build a premium, multi-page, SEO-friendly website that feels handcrafted, loads fast, and drives WhatsApp inquiries.

## Stack rules
- Use Next.js 15 with App Router
- Use TypeScript
- Use Tailwind CSS v4
- Use Google Fonts only
- Use Font Awesome for icons
- Avoid Bootstrap, jQuery, and heavy UI kits
- Keep animations subtle and purposeful
- Prefer static rendering where possible
- Optimize images with next/image
- Keep client-side JavaScript minimal

## UX rules
- No emojis in the UI
- Use a floating WhatsApp button on every page
- Make WhatsApp the primary conversion path
- Use clear sections, generous spacing, and readable typography
- Make the site look premium, not generic or template-based

## Content rules
- Do not invent certifications, awards, or named customer testimonials
- Sample testimonials and sample projects may be used as draft content only
- Rewrite copy so it sounds natural, local, and professional
- Keep claims believable and easy to verify
- Mention Lagos and nationwide service coverage
- Mention 8am to 10pm daily hours
- Mention electrical services, solar, appliances, kitchens, interior design, and laundry setup

## SEO rules
- Every page needs unique title and meta description
- Add local SEO keywords naturally
- Add JSON-LD schema for Organization, LocalBusiness, Service, FAQ, and Breadcrumbs where appropriate
- Include canonical URLs
- Ensure headings are semantically correct
- Include alt text for all meaningful images

## Implementation rules
- Use reusable components
- Keep components small and readable
- Separate content from UI when practical
- Create pages for Home, About, Services, Projects, Gallery, FAQ, and Contact
- Use WhatsApp prefilled message links for CTAs

## Acceptance criteria
- Fast loading
- Clean responsive layout
- Strong mobile experience
- SEO-ready
- Professional and believable copy
- Client can review and approve easily
