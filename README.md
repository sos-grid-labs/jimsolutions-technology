# Jimsolutions Technology Website Starter Pack

This folder is a lightweight planning pack for a clean, fast, multi-page business website.

## Recommended stack
- Next.js 15 App Router
- TypeScript
- Tailwind CSS v4
- Font Awesome
- Google Fonts
- Framer Motion for subtle motion only
- Vercel for deployment
